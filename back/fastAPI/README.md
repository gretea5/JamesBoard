## fastAPI 사용법

### 1. 파이썬 가상환경 생성

### python -m venv venv

### 2. 가상환경 활성화

### source venv/Scripts/activate

### 3. requirements.txt 에서 가상환경 구성

### pip install -r requirements.txt

### 4. fastAPI 실행

### uvicorn main:app --reload
